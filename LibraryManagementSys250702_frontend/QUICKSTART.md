# 快速启动指南

## 前端项目启动步骤

### 1. 安装依赖
```bash
cd LibraryManagementSys250702_frontend
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问应用
打开浏览器访问：http://localhost:5173
